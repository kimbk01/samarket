## 3. 관리자 연동 더블체크

| 항목 | 요구사항 | 구현 위치 | 확인 |
|------|----------|-----------|------|
| **노출 조건** | `user.email === 'im2pact@gmail.com'` | `lib/auth/get-current-user.ts` → `isAdminUser(user)` | ✅ |
| **위치** | 설정 페이지 **최하단** (기타 섹션 아래) | `SettingsList.tsx`에서 4개 그룹 렌더 후 별도 `section`으로 "관리자 접속" 렌더 | ✅ |
| **노출 로직** | 일반 유저: 미노출 / 관리자 이메일: 노출 | `const showAdmin = isAdminUser(user);` 후 `{showAdmin && (...)}` | ✅ |
| **클릭 동작** | `/admin` 이동 | `<Link href="/admin">관리자 접속</Link>` | ✅ |
| **UI** | 별도 카드, 텍스트는 시그니처 색 | `rounded-xl bg-white shadow-sm`, `text-signature font-medium` | ✅ |
| **테스트** | 일반 유저 시 이메일 변경 시 버튼 숨김 | `getCurrentUser()`의 email을 `user@example.com` 등으로 바꾸면 미노출 | ✅ |

**코드 참고**

- 관리자 여부: `lib/auth/get-current-user.ts`  
  `isAdminUser(user)` → `user?.email === 'im2pact@gmail.com'`
- 설정 하단 렌더: `components/mypage/SettingsList.tsx`  
  기타 그룹 다음에 `{showAdmin && (<section data-admin-entry>...</section>)}`

## 4. 참고만 하고 빌더에 미반영한 것

- **알림 배너**: 상단 배너는 별도 컴포넌트로 추후 추가 가능.
- **사장님 라운지**: 비즈니스 사용자용 별도 탭; 현재는 "나의 비즈니스"로 대응.
- **설정 > 앱 설정 / 내 동네 설정 / 고객지원** 등: 별도 탭 구성은 추후 확장.

## 5. 요약

- **나의 카마켓**: 상단, 프로필(매너온도 배지 시그니처), 카마켓페이(시그니처 버튼), 서비스 그리드, 요약 메뉴(가로 3개+혜택 알림점), 나의 거래/관심/활동/비즈니스(아이콘+화살표) 모두 반영됨.
- **설정**: 내 계정, 알림(토글), 서비스 설정, 기타 전 항목 아이콘·우측 값(언어/버전) 반영됨.
- **관리자 접속**: `im2pact@gmail.com`일 때만 설정 **최하단** 별도 카드로 노출, 클릭 시 `/admin` 이동하도록 구현되어 있음.
